# Morning Workflow Protocol

## Formål
Gi rask oversikt over dagens prioriteter, innkommet post og forfallende oppgaver.

## Trigger
Kjør hver morgen senest kl. 08:30, eller når du starter arbeidsdagen.

## Innhold (i denne rekkefølgen)

1. **Dagens dato og tidssone**
   - Aktuelle frister (hvis noen)

2. **Innkommende epost sist 24 timer**
   - Prioritert etter avsender (Calle, Suzan, ledelse først)
   - Handlingsbehov flagget
   - Kan vente til senere

3. **Oppgaver og deadlines**
   - Oppgaver som forfaller i dag
   - Oppgaver som forfaller denne uken
   - Varsling hvis noe er overdue

4. **Dagens timeplan (hvis tilgjengelig)**
   - Møter
   - Blokkert tidsrom

5. **En-linjer for dagens fokus**
   - Maksimum 3 tema

## Format
Bruk punktlister. Kort og slagkraftig.
